A cli tool, easy way to `git init` a new repository.

## Usage

```bash
$ gitinit -h

  hello gitinit

  Usage:
    gitinit (-i|--init) [--remote=<repo>]
    gitinit -b|--bare
    gitinit -h|--help
    gitinit -v|--version

  Options:
    -h --help     show help message
    -v --version  show version
```
